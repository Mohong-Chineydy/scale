📊 9Axes 政治倾向多维分析测试系统

通过195道精心设计的题目，从16个维度深入分析您的政治立场

📋 项目概述

这是一个基于 Bash 的交互式政治倾向测试工具，通过 195道题目 分析用户在 16个维度 上的政治立场。该系统能够深入、全面地评估个人的多维政治观点，提供详细的分析报告。

🎯 核心特性

✨ 195道精心设计的题目

覆盖16个政治维度，全面深入

🎨 实时交互式界面

类似 top 命令的动态刷新效果

📊 多维度分析

同时评估16个独立的政治维度

💾 动态数据统计

实时更新各维度累计分值

🛠️ 灵活的操作方式

支持跳题、退出等多种选项

📈 详细的结果输出

完整的倾向分析和置信度评估

🚀 快速开始

前置要求

✓ Bash 4.0+

✓ Unix/Linux 系统或 macOS

✓ 标准 Linux 命令行工具

安装与运行

# 克隆项目
git clone https://github.com/MoHong-Chineydy/scale.git
cd scale

# 赋予脚本执行权限
chmod +x scripts/9axes-test.sh

# 运行测试
./scripts/9axes-test.sh

📖 使用指南

操作说明

按键 功能
1-5 提交答案（1=非常反对，5=非常同意）
0 或 回车 跳过当前题目
q 退出测试

🧠 16个维度介绍

# 维度名称 说明
1 经济维度 从左翼（高税收、政府干预）到右翼（自由市场、私有化）
2 政治维度 从中央集权到权力分散的政治体制偏好
3 社会维度 从传统/保守到进步/自由的社会价值观
4 权力集中度 对强有力领导和秩序的偏好程度
5 军国主义倾向 对国防军事的态度，从和平主义到军事优先
6 民族主义倾向 国家利益vs全球化，本地化vs国际化
7 宗教信仰度 宗教在社会和政策中的角色
8 进步倾向 对社会变革和创新的接受度
9 传统倾向 对既有制度和习俗的重视程度
10 环保主义 环保优先还是经济发展优先
11 个人主义 个人自由和权利的重要性
12 集体主义 集体利益和社会责任的重要性
13 科学理性 对科学方法和证据的信任度
14 务实主义 注重实际结果vs遵守原则
15 理想主义 对理想和价值观的坚持程度
16 本地化倾向 本地社区vs全球化联系的偏好
