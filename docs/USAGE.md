# 使用指南

## 安装步骤

### 1. 克隆仓库

```bash
git clone https://github.com/MoHong-Chineydy/scale.git
cd scale
```

### 2. 设置权限

```bash
chmod +x scripts/9axes-test.sh
```

### 3. 运行程序

```bash
./scripts/9axes-test.sh
```

#### 数据含义

- **已答题目数**: 已完成的题目数量
- **维度得分**: 该维度的累计得分
- **倾向指示**: 箭头显示左/中/右倾向
- **响应模式**: 用户的答题风格分析
- **完成度**: 测试进度百分比

## 操作方式

### 基础操作

#### 1. 提交答案 (1-5)

```bash
请选择 (1-5, 0-跳过, q-退出)：3
```

按 1-5 之间的任何数字提交您的答案：

- `1` = 非常反对 (强烈左翼/低分)
- `2` = 反对 (温和左翼)
- `3` = 中立 (无明确立场)
- `4` = 同意 (温和右翼)
- `5` = 非常同意 (强烈右翼/高分)

#### 2. 跳过题目 (0 或 回车)

```bash
请选择 (1-5, 0-跳过, q-退出)：0
```

或直接按回车键跳过当前题目。

#### 3. 退出测试 (q)

```bash
请选择 (1-5, 0-跳过, q-退出)：q
感谢使用本问卷。再见！
```

## 得分解释

### 单维度得分

每个维度的得分范围是从 -30 到 +30（因为每道题 ±2 分，共15道题）。

| 分数范围 | 倾向 | 符号 | 含义 |
|---------|------|------|------|
| < -8 | 极端左翼 | ▼▼ | 强烈左翼立场 |
| -8 ~ -4 | 左翼倾向 | ▼ | 温和左翼立场 |
| -4 ~ +4 | 中立立场 | ◄► | 没有明确立场 |
| +4 ~ +8 | 右翼倾向 | ▲ | 温和右翼立场 |
| > +8 | 极端右翼 | ▲▲ | 强烈右翼立场 |

### 响应模式分类

系统根据您的回答模式分析您的思维风格：

#### 极端化立场
- 特点：超过60%的回答为1或5（两端）
- 含义：您对问题有非常明确和坚定的看法
- 建议：可能需要更多思考来理解其他观点

#### 明确而坚定的立场
- 特点：40-60%的回答为1或5
- 含义：您有清晰的政治观点但也愿意考虑其他可能性
- 建议：良好的平衡状态

#### 适度和均衡
- 特点：20-40%的回答为1或5
- 含义：您在许多问题上都有中等强度的看法
- 建议：正常的政治思维模式

#### 谨慎/中立倾向
- 特点：40-60%的回答为3（中立）
- 含义：您对许多问题保持中立或不确定
- 建议：可能需要更多信息或思考某些问题

#### 极端谨慎/高度中立
- 特点：超过60%的回答为3
- 含义：您在大多数问题上都无法确定或采取中立立场
- 建议：深入思考您的真实立场

## 常见问题

### Q: 我可以重新回答某个问题吗？
A: 不行。当前版本不支持回溯修改。但您可以重新开始测试。

### Q: 我可以保存结果吗？
A: 目前结果显示在屏幕上。您可以截屏保存或手动记录。

### Q: 测试需要多长时间？
A: 平均15-20分钟，取决于您的思考速度。

### Q: 我应该如何理解我的分数？
A: 详见 `DIMENSIONS.md` 中的各维度解释。

### Q: 这个测试的准确性如何？
A: 这是一个相对主观的政治立场评估工具，不是科学诊断。
